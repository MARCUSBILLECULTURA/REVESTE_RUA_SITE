# ReVeste Rua — site em React + Vite

Projeto institucional do ReVeste Rua, pronto para publicação em Vercel ou Netlify.

## Instalação

```bash
npm install
```

## Rodar localmente

```bash
npm run dev
```

## Gerar versão final

```bash
npm run build
```

A pasta pronta para publicação será criada em:

```bash
dist/
```

## Publicação no Vercel

1. Envie esta pasta para um repositório no GitHub
2. Crie uma conta no Vercel
3. Clique em **Add New Project**
4. Conecte o repositório
5. Clique em **Deploy**

## Publicação no Netlify

### Opção 1
Conecte o repositório do GitHub e publique direto.

### Opção 2
Rode `npm run build` e envie a pasta `dist/` no painel do Netlify.
